# Cursor Landing Page Clone

This repository contains a static clone of the Cursor AI landing page, built using pure HTML and CSS.  
The goal of this project is to practice modern landing page layouts, responsive design, and clean UI structuring without using any frameworks.

## Preview

This project recreates the overall look and feel of the Cursor website, including:

- Sticky navigation header  
- Hero section with layered images  
- Partner logos grid  
- Feature sections with image overlays  
- Testimonials section  
- Changelog and blog highlights  
- Footer with multiple navigation columns  

## Tech Stack

- HTML5  
- CSS3 (Flexbox + Grid)  
- No frameworks  
- No JavaScript

## Key Features

- Fully static landing page  
- Sticky header navigation  
- Responsive layouts using CSS Grid and Flexbox  
- Layered image sections using absolute positioning  
- Clean typography and spacing  
- Dark theme inspired by Cursor’s design  
- Well-structured semantic HTML  

## Sections Breakdown

- Header  
  Logo, navigation links, and call-to-action buttons with sticky behavior.

- Hero Section  
  Main headline and primary download CTA.

- Image Showcase  
  Overlapping screenshots using relative and absolute positioning.

- Partners Section  
  Grid-based layout showcasing trusted companies.

- Feature Sections  
  Alternating text and image layouts explaining product features.

- Testimonials  
  Card-based testimonial layout with user avatars.

- Frontier & Changelog  
  Highlights product updates, models, and new features.

- Career & Blog  
  Team showcase and recent blog highlights.

- Footer  
  Multi-column footer with product, company, legal, and social links.

## How to Run Locally

2. Open the project folder
3. Open `index.html` directly in your browser  
(no server required)
